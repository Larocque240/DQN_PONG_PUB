* Description
*    This program is used to the classic video game pong.
*	 Pong is a two-dimensional sports game that simulates table tennis. 
*	 The player controls an in-game paddle by moving it vertically across 
*	 the left or right side of the screen.  
*	 The goal is for each player to reach eleven points before the opponent; 
*	 points are earned when one fails to return the ball to the other.
*	 The twist was the addition of multiple balls
*
